🚀 SETUP (3 STEPS - 10 MINUTES)
═══════════════════════════════════════════════════════════════

STEP 1: Get YouTube API Key
───────────────────────────────────────────────────────────────
1. Go to: https://console.cloud.google.com/
2. Create a new project (or select existing)
3. Enable "YouTube Data API v3"
4. Create Credentials → API Key
5. Copy the key (looks like: AIzaSy...)

📖 Detailed instructions: See YOUTUBE_SETUP.md


STEP 2: Configure the Key
───────────────────────────────────────────────────────────────
1. Open file: youtube_sync.php
2. Find line 11:
   define('YOUTUBE_API_KEY', 'YOUR_API_KEY_HERE');
   
3. Replace YOUR_API_KEY_HERE with your actual key:
   define('YOUTUBE_API_KEY', 'AIzaSyB123456...');

4. Save the file


STEP 3: Test It!
───────────────────────────────────────────────────────────────
1. Test connection:
   http://localhost/.../youtube_sync.php?action=test

2. Sync your channel:
   http://localhost/.../youtube_sync.php?action=sync

3. View in dashboard:
   Open communication.html → Social Media tab
   Click "Sync Now" button on YouTube card
