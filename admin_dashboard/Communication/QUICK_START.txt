🚀 SETUP IN 3 STEPS
═══════════════════════════════════════════════════════════════════════

STEP 1: Import Database Schema
────────────────────────────────────────────────────────────────────────
1. Make sure XAMPP MySQL is running
2. Open phpMyAdmin: http://localhost/phpmyadmin
3. Select database: church_management
4. Click the "SQL" tab
5. Open: social_media_setup.sql
6. Copy entire contents and paste into SQL tab
7. Click "Go" button
8. ✅ You should see: "3 tables created successfully"

🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════

Problem: "Database connection failed"
Solution: 
  - Make sure MySQL is running in XAMPP
  - Check database name is "church_management"
  - Verify credentials in db_connect.php

Problem: "Social media not loading"
Solution:
  - Press F12 to open browser console
  - Check for JavaScript errors
  - Verify tables exist in phpMyAdmin
  - Clear browser cache (Ctrl+Shift+Delete)

Problem: "Metrics showing 0"
Solution:
  - Run social_media_setup.sql again
  - Check INSERT statements executed
  - Use test_social_api.php to verify data
